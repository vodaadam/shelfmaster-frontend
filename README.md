# Shelf master frontend

Frontend aplikace postavená na **Vue** (spouštění přes `npm`).

---

## Požadavky

- **Node.js** (doporučeno LTS)
- **npm** (součást Node.js)

Ověření:
```bash
node -v
npm -v
```

---

## Instalace závislostí

V kořeni projektu:
```bash
npm install
```

---

## Spuštění ve vývoji

Spustí vývojový server (hot-reload):
```bash
npm run dev
```
